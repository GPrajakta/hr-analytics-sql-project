#  HR Analytics SQL Project

##  Project Overview

This project analyzes an HR dataset using SQL to generate actionable business insights related to employee distribution, salary trends, department performance, and workforce analytics.

---

##  Objectives

* Analyze workforce demographics
* Identify salary distribution trends
* Evaluate department performance
* Solve real-world business problems using SQL
* Build a strong portfolio for Data Analyst roles

---

##  Dataset

The dataset contains the following tables:

* employees
* departments
* salaries
* titles
* dept_emp
* dept_manager

---

##  Tools & Technologies

* SQL (PostgreSQL / MySQL)
* Power BI / Tableau
* Python (Pandas for EDA)
* Git & GitHub

---

##  Key Analysis Performed

###  Basic Analysis

* Employee count by department
* Gender distribution
* Hiring trends

###  Intermediate Analysis

* Average salary by department
* Top-paid employees
* Department-wise employee growth

###  Advanced Analysis

* Window functions (RANK, DENSE_RANK)
* Running totals and cumulative analysis
* Z-score salary analysis
* Employee retention trends

###  Business Questions Solved

* Which department has the highest salary expense?
* Who are the top 10 highest-paid employees?
* What is the attrition trend?
* Which roles are most critical?

##  How to Run

1. Clone the repository

```
git clone https://github.com/your-username/hr-analytics-sql-project.git
```

2. Load data into PostgreSQL/MySQL

3. Run SQL scripts:

```
sql/schema/create_tables.sql
sql/data_loading/load_data.sql
sql/analysis/
```
